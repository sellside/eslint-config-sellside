> {%= description %}

## Install

Install to `devDependencies` with [npm](https://www.npmjs.com/):

```sh
$ npm install @sellside/eslint-config-sellside -D
```

## Usage

Add the following property to package.json:

```json
{
  "eslintConfig": {
    "extends": "@sellside/sellside"
  }
}
```
